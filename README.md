# Task Manager Frontend

React frontend for the Task Manager API.

## Features
- User authentication (JWT)
- View tasks
- Create tasks
- Mark tasks complete
- Filter by status and priority

## Tech Stack
- React 18
- Vite
- Axios
- React Router (coming soon)

## Setup
```bash
npm install
npm run dev
```

## Environment Variables

Create `.env` file:
```
VITE_API_URL=http://54.80.178.193:8000
```

## Backend
